Sombras

Con HTML5 podemos crear de una manera sencilla tanto sombras para zonas del HTML (< div>, < article>, < footer>, < header>...), como para textos. Aunque cada uno de estos tipos de sombra tiene su propia etiqueta, la sintaxis sigue la misma lógica. Así, en el caso de la sombra aplicada a < div>, zonas o secciones del html utilizamos la propiedad box-shadow.
